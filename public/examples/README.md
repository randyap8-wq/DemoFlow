# Example demos

Drop these JSON files into the **Load File** picker (or pass them via
`?demo=examples/<filename>`) to see DemoFlow in action.

| File                | Description                                       |
| ------------------- | ------------------------------------------------- |
| `welcome.json`      | A two-step welcome tour with a single hotspot.    |
| `landing-page.html` | A raw HTML landing page rendered as a single step. |
